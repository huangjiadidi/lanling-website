# 公司开发者账号 + 国内上架准备清单

把这份当 checklist,做完一项勾一项。**邮箱 ai@lanling.vip 已经满足苹果对"公司域名邮箱"的要求,这是个隐藏门槛,你已经过了。**

## 一、立即开始(Day 1,全部并行)

### ☐ 邓白氏编码 (D-U-N-S Number)
苹果开发者账号最慢的卡点,务必最先做。

- 入口:https://developer.apple.com/enroll/duns-lookup/
- 先用公司中英文名查询,可能已经有(很多公司注册时被默认登记过)
- 如果没有,在线填表申请
- 信息要和**营业执照 100% 一致**,包括标点、空格、英文翻译
- 时间:5–14 工作日,免费
- 完成标志:邮件收到 9 位数字的 D-U-N-S 号

### ☐ 阿里云大陆轻量服务器
ICP 备案的硬性前提,必须先买。

- 阿里云控制台 → 轻量应用服务器
- 地域:华东/华北/华南任选,**不能选香港或海外**
- 规格:2核2G/3M 带宽够用,约 ¥60–80/月
- 时长:**至少 3 个月**,否则不让备案
- 系统:Ubuntu 22.04 或 CentOS 都行

### ☐ 占位站托管到 GitHub Pages
Apple 申请页面要填"官网"URL,这一步当天就能搞定。

- 详细步骤见同目录 `deploy.md`
- 完成标志:`lanling.vip` 或 `xxx.github.io/lanling-website/` 能打开

### ☐ 提交 ICP 备案
- 入口:阿里云控制台 → 备案 → 新增备案
- 准备材料:
  - 法人身份证正反面照片
  - 营业执照副本照片
  - 法人手持身份证照片
  - 公司对公电话(座机或可接通的手机)
  - 网站负责人信息
  - 阿里云会**免费寄"备案幕布"** 到你的办公地址,收到后拍主体人员照片上传
- 时长:7–20 工作日
- 完成标志:工信部短信通知 + 阿里云后台显示备案号(如 `沪ICP备2026000000号`)

### ☐ 软件著作权登记 × 2
安卓上架硬性要求,iOS 强烈建议有。

- 蓝领智工 软著一份
- 门安 软著一份
- 官方入口:https://register.ccopyright.com.cn(自然受理 30 工作日)
- 加急渠道:淘宝搜"软著加急",3–7 工作日,约 ¥1500/个
- 完成标志:拿到《计算机软件著作权登记证书》PDF + 纸质证书

## 二、邓白氏码下来后(第 1–2 周)

### ☐ 注册苹果开发者账号
- 入口:https://developer.apple.com/programs/enroll/
- 用 **ai@lanling.vip** 创建 Apple ID(✅ 已经是公司域名邮箱,符合要求)
- 在 Apple ID 上开启**双重认证**(必需,否则不让申请)
- 选 **Organization / Company**,不要选 Individual
- 填入邓白氏码,系统自动带出公司信息
- 填法人姓名(Authorized Contact,必须是邓白氏注册的联系人)
- 等待 Apple 回拨公司电话(英文或中文都可能,提前准备)
- 通过后用信用卡支付 **$299/年**(法人本人的 Visa/Master 信用卡)
- 时长:1–4 周
- ⚠️ 重要:接 Apple 电话的人必须是邓白氏登记的 Authorized Contact 本人

## 三、ICP 备案通过后(第 3–5 周)

### ☐ 把网站迁回国内服务器(或继续 GitHub Pages)
- 阿里云 DNS 把 `lanling.vip` 解析到轻量服务器 IP
- 服务器装 Nginx,把 `lanjing-website/` 目录传到 `/var/www/html/`
- 浏览器访问 `lanling.vip` 确认能开

### ☐ 网站底部填入 ICP 备案号
查找替换所有 HTML 文件里的 `ICP 备案号:待办理`:
```
沪ICP备2026000000号  (示例,改成你的真实备案号)
```

### ☐ 公安备案(ICP 通过后 30 天死线!)
经常被忘,漏做要罚款 + 关站。

- 入口:https://beian.mps.gov.cn/
- 注册账号 → 新办备案 → 主体备案 → 网站备案
- 上传材料和 ICP 备案类似
- 时长:5–20 工作日
- 完成后,把公安备案号也填到网站底部

### ☐ APP 备案 × 2(必须在 ICP 主体备案通过后)
- 阿里云控制台 → APP 备案 → 新增
- 复用已有的主体备案(不用重新提交身份信息)
- 每个 App 单独提交:
  - **蓝领智工**:Bundle ID `com.lanjing.lanlingzhigong` + 安卓签名 MD5
  - **门安**:Bundle ID(待定)+ 安卓签名 MD5
- App 提供服务的域名填 `lanling.vip`
- 时长:7–20 工作日
- 完成标志:工信部 APP 备案号,格式如 `沪ICP备2026000000号-1A`

## 四、上架(第 6–10 周)

### ☐ iOS 上架 App Store(中国区)
- App Store Connect → My Apps → 新建 App
- 必填项:
  - APP 备案号 ⚠️ 中国区强制
  - 隐私政策 URL → `https://lanling.vip/privacy.html`
  - 用户协议 URL → `https://lanling.vip/terms.html`
  - 截图、描述、关键词、年龄分级
- 提交审核,1–7 天

### ☐ 安卓上架(每家应用市场单独提)
- 华为应用市场:https://developer.huawei.com/
- 小米应用商店:https://dev.mi.com/
- OPPO 应用商店:https://open.oppomobile.com/
- vivo 应用商店:https://dev.vivo.com.cn/
- 应用宝(腾讯):https://open.tencent.com/
- 每家需要:开发者账号(企业认证) + 软著证书 + APP 备案号 + 隐私政策 URL
- 注册各家开发者账号:免费 ~ ¥300/年不等,需要企业认证打款验证

## 七、最容易踩的坑

1. **邓白氏信息和营业执照不一致** → Apple 审核直接 reject,重做要再等 5–14 天。提交前用 PDF 把营业执照对照检查每个字符。
2. **苹果回拨电话没人接 / 不是 Authorized Contact 接的** → 申请失败重排队。提前告知前台/法人秘书"近期会接到 Apple 国际电话"。
3. **漏做公安备案** → ICP 通过后 30 天死线,过期被关站还要罚款,日历上设个提醒。
4. **APP 备案的"服务器地址"乱填** → 必须是已备案的域名(`lanling.vip`),所以网站备案要走在 APP 备案前面。
5. **个人 Apple 账号下做的 App 想转到公司账号** → 不能转,只能重新上架。一开始就用公司账号。
